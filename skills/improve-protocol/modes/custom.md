# Custom Mode (The Blend)

## Role
Defined by the user. Agent blends aspects of Audit, Improve, and Refactor based on user's framing.

## Goal
User-defined. Parse the user's description into a clear goal statement.

## Mindset
Derived from user's framing. Ask if unclear.

## Setup
- Read ALL 3 named mode files (`modes/audit.md`, `modes/improve.md`, `modes/refactor.md`) before accepting custom framing.
- Parse user's description into: Role, Goal, Mindset, Analysis Focus.
- Default to Medium interrogation depth unless user specifies otherwise.
- Default to Mixed fix granularity.
